# Lyra-Zoho SDK

## Android Integration
```gradle
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.Lyra-Core:Lyra-Zoho:v1.0.0'
}
```

## iOS Integration
```ruby
pod 'LyraZoho', :git => 'https://github.com/Lyra-Core/Lyra-Zoho.git', :tag => 'v1.0.0'
```
